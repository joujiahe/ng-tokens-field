## ng-tokens-field
A simple tokens field for AngularJS

### Requirements
* **AngularJS v1.2.0+**

### Usage
```html
<link href="tokens-field.css" media="all" rel="stylesheet" type="text/css">
<script src="tokens-field.js"></script>

...
<!-- tags must be an array-->
<ng-tokens-field tokens="tags"></ng-tokens-field>
```
